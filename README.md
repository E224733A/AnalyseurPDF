# AnalyseurPDF
Projet d'une petite application en Python qui permet de :  charger un document technique PDF, extraire son texte, repérer automatiquement des informations utiles, faire une recherche intelligente dans le document, générer un résumé structuré. 

Technologies utilisé

Backend / logique

Python

Interface

Streamlit

Vérifier et améliorer au possible : 

Bibliothèques utiles : 

PyMuPDF ou pdfplumber pour lire les PDF

re pour les expressions régulières

scikit-learn pour extraire des mots-clés simples

sentence-transformers pour la recherche sémantique

faiss ou stockage simple en mémoire pour comparer les passages

pandas pour structurer les résultats
